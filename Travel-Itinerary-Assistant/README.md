#  Travel Itinerary Assistant

## Description
An AI-powered assistant that generates personalized travel itineraries based on user preferences like destination, budget, and duration.

---

##  Features
- Generates day-wise travel plans
- Suggests places, food, and activities
- Customizable based on user input
- Fast and interactive responses

---

## Tech Stack
- Python
- OpenAI API
- Prompt Engineering

---

##  Use Case
Helps users plan trips quickly without manual research.
